# Оценка экономической эффективности мероприятий по снижению загрязнения атмосферного воздуха

Streamlit-приложение для реализации **МР 5.1.0158-19** в рамках федерального проекта **«Чистый воздух»**.

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой на Streamlit Cloud

1. Форкните или загрузите репозиторий на GitHub
2. Зайдите на [share.streamlit.io](https://share.streamlit.io)
3. Нажмите **New app** → выберите репозиторий → `app.py`
4. Нажмите **Deploy**

## Методика

Расчёт выполняется в соответствии с МР 5.1.0158-19 «Оценка экономической эффективности реализации мероприятий по снижению уровней загрязнения атмосферного воздуха на основании оценки риска здоровью населения».

**Показатели:**
- `ΔG` — снижение случаев преждевременной смерти
- `ΔR` — снижение случаев заболеваний  
- `ΔHI` — уменьшение индекса опасности
- `ΔCR` — снижение канцерогенного риска
- `ΔY` — предотвращённый экономический ущерб
- `C` — затраты на реализацию программы
